## 📋 功能介绍

- 📊 **学习进度追踪**：可视化展示29课次的学习进度
- ⏰ **倒计时**：距离考试的天数提醒
- ✅ **状态管理**：已完成/进行中/未开始
- 💾 **本地保存**：进度自动保存到浏览器localStorage
- 📱 **PWA支持**：可添加到手机桌面使用

## 🛠️ 技术栈

- 纯HTML/CSS/JavaScript，无需后端
- PWA渐进式Web应用
- localStorage本地存储
